## Getting Started

clone the repo

```bash
git clone https://github.com/sengayire/youtube-clone.git
```

go to the directory

```bash
cd youtube-clone
```

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
